# creche_sucePouce

creche_sucePouce est un site internet présentant des images/actualités

### Pré-requis

* PHP 8.1
* Composer
* Symfony cli
* Docker
* nodejs et npm

### Lancer l'environement de dévelopement

composer install
npm install
npm run build
symfony server:start
## Lancer des tests
’’’bash
php bin/phpunit --testdox
’’’
